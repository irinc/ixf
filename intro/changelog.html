Current Release - 1.1.2 - March 19, 2012

  • Fixed problem (when using jQuery 1.6+) with ixf-autocomplete not populating the alternate input (referenced by the "data-ac-alternate-id" attribute of the "ixf-autocomplete" input) and you click on an item from the drop-down.
  • Fixed problem (when using jQuery 1.6+) with "data-layout-minSize", "data-layout-maxSize", and "data-layout-initClosed" attributes for layout configuration.
  • Upgraded jQuery.layout from version 1.3.0.rc29.15 to 1.3.0.rc30.3

Previous Updates

1.1.1 - March 14, 2012

  • Upgraded jQuery.dataTables to version 1.9.0.
  • Fixed Tables so that input[type=text] filter fields are pre-populated with saved values in the session cookie (if the bStateSave=true option is used)
  • Updated "Column Filtering" demo to use bStateSave=true and document the fact that if you want sort column headers AND column filter headers, set data-dt-options='{"bSortCellsTop":true}' in the table tag.
